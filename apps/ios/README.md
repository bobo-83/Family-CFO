# iOS App

Planned technology: SwiftUI.

Responsibilities:

- Chat interface
- Camera and receipt capture
- Store item capture
- Face ID
- Notifications
- QR pairing with the home server
- Secure local authentication

The iPhone app should not perform financial reasoning. It may perform lightweight image understanding through Apple-supported frameworks and send structured JSON to the home server.

Implementation is pending the mobile specification in `docs/specs/08-mobile-spec.md`.
